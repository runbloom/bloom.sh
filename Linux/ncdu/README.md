# What is ncdu

It is a terminal based tool which scans your directories.

Then it ranks all of them by their size.

It is the most effective way to see what exactly is taking up space.

You also delete any files/folders you'd like.

<br>
<br>

# How to download
To install the software simply run

*Change the install command based on your distro*
```bash
sudo dnf install ncdu
```

<br>
<br>

# How to use
Scan your whole system

```bash
ncdu /
```
You can move through your folders via arrow keys

<br>

## Keybinds
~ To delete: 'd'

~ Toggle percentage/bars/both: 'g'

~ To quit: 'q'
